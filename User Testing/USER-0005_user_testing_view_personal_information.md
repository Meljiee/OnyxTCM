## **USER-0005:** User testing - View Personal Information
> **Summary:** Verify that a user can navigate to their profile and view personal details. <br>

**Preconditions:** User must be logged in.

**Scenario 1**

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | On any main dashboard screen, tap the `Profile` icon (user silhouette) in the top right appbar. | Verify that the `Profile` screen is shown, defaulting to the `Personal` tab. |
| 2 | Observe the `Personal Information` section. | Verify that the user's avatar, full name, email, First Name, Last Name, and Preferred Name are displayed correctly. |

**Post-conditions:**
- The user can view their account information.
