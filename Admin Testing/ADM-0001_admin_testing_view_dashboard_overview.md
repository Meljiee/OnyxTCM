## **ADM-0001:** Admin testing - View Dashboard Overview
> **Summary:** Verify that the admin can view system performance and statistics on the main dashboard. <br>

**Preconditions:** Admin user must be logged in.

**Scenario 1**

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | In the sidebar, tap `Dashboard` menu item. | Verify that the `Dashboard Overview` screen is shown. |
| 2 | Observe the top statistics cards. | Verify that `Total Customers Today`, `Completed Services`, `Average Waiting Time`, and `Current Queue Length` display numerical values. |
| 3 | Observe the `Weekly Queue Volume` chart. | Verify that a bar chart displaying queue volume from Monday to Sunday is visible and populated. |

**Post-conditions:**
- The admin successfully views real-time system metrics.
