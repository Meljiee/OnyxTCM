## **SEC-0003:** Security testing - Access Security Settings
> **Summary:** Verify that a user can access security settings to change their password or delete their account. <br>

**Preconditions:** User must be logged in.

**Scenario 1**

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Navigate to the `Profile` screen. | Verify that the profile navigation tabs (`Personal`, `History`, `Security`) are visible. |
| 2 | Tap the `Security` tab. | Verify that the `Security` section is shown. |
| 3 | Observe the available actions. | Verify that the `Change Password` and `Delete Account` buttons are visible and accessible. |

**Post-conditions:**
- The user can access account security management tools.
